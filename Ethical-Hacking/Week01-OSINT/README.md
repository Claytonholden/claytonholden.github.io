# Week 01 – OSINT & Reconnaissance

## Objectives
- Understand passive vs active reconnaissance
- Use open-source intelligence tools
- Perform DNS and username enumeration
- Identify exposed services and metadata

---

## Key Concepts

### Passive Recon
Collecting information without directly interacting with the target system.

Examples:
- WHOIS lookups
- DNS records
- Public GitHub repositories
- Social media analysis

### Active Recon
Directly interacting with a target system.

Examples:
- Nmap scanning
- Service enumeration
- Port scanning

---

## Important Terms

- OSINT – Open Source Intelligence
- Footprinting – Mapping the attack surface
- Enumeration – Extracting detailed system info
- Banners – Service/version identifiers

---

## Exam Tip
If a question asks:
- “Gather information without alerting the target” → Passive Recon
- “Identify open services” → Nmap
- “Find domain ownership” → WHOIS
