Ethical Hacking – Weekly Notes & Labs

An 8-week build of practical offensive security fundamentals: OSINT, scanning & enumeration, traffic analysis, log forensics, web app testing, wireless basics, and crypto foundations — plus NCL Gymnasium practice write-ups.

Kali Linux Nmap / NSE Wireshark / Tshark Log Analysis NCL Practice

Course Outcomes

By the end of this project I can:

  • Perform reconnaissance (passive + active) and document an attack surface clearly.
  • Scan and enumerate services with Nmap + NSE scripts and summarize risk findings.
  • Analyze packet captures (HTTP/TLS/Telnet/CAN/wireless basics) with Wireshark/tshark.
  • Extract facts from logs (auth, FTP, web access) using filters, regex, and scripting.
  • Write clean walkthroughs that explain tools, commands, and why the results matter.
