# Topic 3 — Network Exploitation, Traffic Analysis & Reporting

## Core workflow (PCAP/logs)
1. Identify protocol(s)
2. Identify endpoints
3. Extract key artifacts (credentials, commands, files, flags)
4. Validate with filters + streams
5. Summarize timeline + evidence + mitigations

## Tools
- Wireshark: Follow TCP stream, Conversations, Endpoints, Export Objects
- tshark: fast field extraction and filtering
- Python: parsing logs/CSV/hex conversions

## Portfolio bullets
- Comfortable extracting evidence from packet captures and logs.
- Can present findings as a timeline with recommended fixes.
