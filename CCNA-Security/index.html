CCNA Security / Network Defense – Capstone

CYBR-2102 focused on building and defending small enterprise networks: designing defense-in-depth, analyzing packet captures, and writing a full incident response plan.

Cisco / Packet Tracer Wireshark NIST 800-61

Course Outcomes

By the end of the class I could:

  • Design segmented networks with external, DMZ, and internal zones.
  • Write and verify Cisco IOS ACLs enforcing least privilege between those zones.
  • Capture and analyze traffic in Wireshark (ping sweeps, SYN scans, DoS-style floods).
  • Map findings to CIS / NIST controls and recommend concrete defensive actions.
  • Write an incident response plan that follows the full NIST SP 800-61 lifecycle.

Milestones

Milestone 1

Defense-in-Depth Network Design

Packet Tracer design with external, firewall, DMZ, and internal zones. ACLs control web access to the DMZ, restrict management to admin hosts, and block direct DMZ ↔ internal traffic. Local firewalls and an access-control matrix tie network rules back to user roles.

Milestone 2

Traffic Analysis Report (Wireshark)

Captured live traffic and identified ping sweeps, SYN scans, and high-rate ICMP floods. Used Wireshark filters to confirm attacker behavior and mapped countermeasures to CIS Controls and NIST guidance.

Milestone 3

Incident Response Plan

Full NIST SP 800-61 incident lifecycle for a brute-force RDP and data exfiltration scenario: preparation, detection/analysis, containment, eradication, recovery, and lessons learned.
